1. Recursividad

La recursividad es una técnica de programación en la cual una función se llama a sí misma para resolver un problema. Es útil para problemas que pueden dividirse en subproblemas más pequeños y similares.

2. Definición

En programación, la recursividad se define como el proceso en el cual una función se invoca a sí misma directamente o a través de otra función. La recursividad debe tener un caso base para evitar bucles infinitos.

3. Procedimientos recursivos

Los procedimientos recursivos son algoritmos que se resuelven dividiendo el problema en casos más pequeños, resolviendo cada uno de ellos y combinando los resultados. Ejemplos incluyen el cálculo de factoriales, la búsqueda binaria y la resolución de problemas de torres de Hanoi.

4. Ejemplos de casos recursivos

Algunos ejemplos comunes de recursividad incluyen:

  • Cálculo de factoriales: factorial(n) = n * factorial(n-1).
  • Serie de Fibonacci: fibonacci(n) = fibonacci(n-1) + fibonacci(n-2).
  • Búsqueda binaria en listas ordenadas.
  • División de un problema en subproblemas (divide y vencerás).

Ejemplo en Consola


public class Factorial {
    // Método recursivo para calcular el factorial de un número
    public static int factorial(int n) {
        if (n == 0) {
            return 1;
        } else {
            return n * factorial(n - 1);
        }
    }

    public static void main(String[] args) {
        int num = 5;
        int result = factorial(num);
        System.out.println("El factorial de " + num + " es " + result);
    }
}
Resultado esperado: 
El factorial de 5 es 120
