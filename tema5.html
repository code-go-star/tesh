Métodos de ordenamiento

Temario

5 Métodos de ordenamiento

5.1 Algoritmos de ordenamiento internos

  • 5.1.1 Burbuja

    • El método de ordenamiento burbuja es uno de los algoritmos de ordenamiento más simples. Compara pares de elementos adyacentes y los intercambia si están en el orden incorrecto.

  • 5.1.2 Quicksort

    • Quicksort es un algoritmo de ordenamiento eficiente que utiliza el principio de divide y vencerás. Divide el arreglo en subarreglos más pequeños y los ordena recursivamente.

  • 5.1.3 ShellSort

    • ShellSort es una variante del ordenamiento por inserción que permite intercambiar elementos que están lejos unos de otros para reducir el número total de intercambios.

  • 5.1.4 Radix

    • El ordenamiento Radix ordena números (o cadenas) considerando cada dígito o carácter en su valor posicional. Es eficiente para datos de longitud fija.

5.2 Algoritmos de ordenamiento externos

  • 5.2.1 Intercalación

    • La intercalación es un método de ordenamiento externo que fusiona varios archivos ordenados en uno solo. Se usa comúnmente en la ordenación de grandes volúmenes de datos.

  • 5.2.2 Mezcla Directa

    • La mezcla directa ordena los datos dividiéndolos en bloques pequeños, ordenando cada bloque y luego fusionándolos. Es útil cuando los datos no caben en memoria principal.

  • 5.2.3 Mezcla Natural

    • La mezcla natural es una técnica que identifica subsecuelas ya ordenadas en los datos y las fusiona. Reduce la cantidad de pases necesarios para ordenar los datos.

Ejemplo en Java


import java.util.Arrays;

public class MetodosDeOrdenamiento {

    // Método de ordenamiento Burbuja
    public static void burbuja(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n-1; i++) {
            for (int j = 0; j < n-i-1; j++) {
                if (arr[j] > arr[j+1]) {
                    int temp = arr[j];
                    arr[j] = arr[j+1];
                    arr[j+1] = temp;
                }
            }
        }
    }

    // Método de ordenamiento Quicksort
    public static void quicksort(int[] arr, int low, int high) {
        if (low < high) {
            int pi = partition(arr, low, high);
            quicksort(arr, low, pi-1);
            quicksort(arr, pi+1, high);
        }
    }

    private static int partition(int[] arr, int low, int high) {
        int pivot = arr[high];
        int i = (low-1);
        for (int j = low; j < high; j++) {
            if (arr[j] < pivot) {
                i++;
                int temp = arr[i];
                arr[i] = arr[j];
                arr[j] = temp;
            }
        }
        int temp = arr[i+1];
        arr[i+1] = arr[high];
        arr[high] = temp;
        return i+1;
    }

    // Método de ordenamiento ShellSort
    public static void shellSort(int[] arr) {
        int n = arr.length;
        for (int gap = n/2; gap > 0; gap /= 2) {
            for (int i = gap; i < n; i++) {
                int temp = arr[i];
                int j;
                for (j = i; j >= gap && arr[j - gap] > temp; j -= gap) {
                    arr[j] = arr[j - gap];
                }
                arr[j] = temp;
            }
        }
    }

    // Método de ordenamiento Radix
    public static void radixSort(int[] arr) {
        int m = getMax(arr);
        for (int exp = 1; m/exp > 0; exp *= 10) {
            countSort(arr, exp);
        }
    }

    private static void countSort(int[] arr, int exp) {
        int n = arr.length;
        int[] output = new int[n];
        int[] count = new int[10];
        Arrays.fill(count, 0);

        for (int i = 0; i < n; i++) {
            count[(arr[i] / exp) % 10]++;
        }

        for (int i = 1; i < 10; i++) {
            count[i] += count[i - 1];
        }

        for (int i = n - 1; i >= 0; i--) {
            output[count[(arr[i] / exp) % 10] - 1] = arr[i];
            count[(arr[i] / exp) % 10]--;
        }

        for (int i = 0; i < n; i++) {
            arr[i] = output[i];
        }
    }

    private static int getMax(int[] arr) {
        int max = arr[0];
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] > max)
                max = arr[i];
        }
        return max;
    }

    public static void main(String[] args) {
        int[] arr = {170, 45, 75, 90, 802, 24, 2, 66};
        System.out.println("Arreglo original: " + Arrays.toString(arr));

        burbuja(arr);
        System.out.println("Ordenado por Burbuja: " + Arrays.toString(arr));

        int[] arr2 = {170, 45, 75, 90, 802, 24, 2, 66};
        quicksort(arr2, 0, arr2.length - 1);
        System.out.println("Ordenado por Quicksort: " + Arrays.toString(arr2));

        int[] arr3 = {170, 45, 75, 90, 802, 24, 2, 66};
        shellSort(arr3);
        System.out.println("Ordenado por ShellSort: " + Arrays.toString(arr3));

        int[] arr4 = {170, 45, 75, 90, 802, 24, 2, 66};
        radixSort(arr4);
        System.out.println("Ordenado por Radix: " + Arrays.toString(arr4));
    }
}
