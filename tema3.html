3 Estructuras Lineales

3.1 Pilas

  • 3.1.1 Representación en memoria

    • Las pilas suelen representarse mediante arreglos o listas enlazadas, donde la última posición insertada es la primera en salir (LIFO).

  • 3.1.2 Operaciones básicas

    • Las operaciones básicas incluyen push (insertar), pop (eliminar) y peek (consultar el elemento en la cima).

  • 3.1.3 Aplicaciones

    • Las pilas se utilizan en la gestión de memoria, en la implementación de algoritmos de reversión y en la evaluación de expresiones.

3.2 Colas

  • 3.2.1 Representación en memoria

    • Las colas pueden representarse mediante arreglos o listas enlazadas, donde el primer elemento insertado es el primero en salir (FIFO).

  • 3.2.2 Operaciones básicas

    • Las operaciones básicas incluyen enqueue (insertar) y dequeue (eliminar).

  • 3.2.3 Tipos de colas

    • Existen varios tipos de colas, como colas simples, colas circulares y bicolas.

  • 3.2.4 Aplicaciones

    • Las colas se utilizan en la gestión de procesos en sistemas operativos y en algoritmos de búsqueda en anchura.

3.3 Listas

  • 3.3.1 Operaciones básicas

    • Las operaciones básicas incluyen inserción, eliminación y recorrido de elementos.

  • 3.3.2 Tipos de listas

    • Existen diferentes tipos de listas, como listas simplemente enlazadas, doblemente enlazadas y circulares.

  • 3.3.3 Aplicaciones

    • Las listas se utilizan en estructuras de datos dinámicas y en la implementación de otros tipos de datos abstractos.

Ejemplo en Java


import java.util.LinkedList;
import java.util.Queue;
import java.util.Stack;

public class EstructurasLineales {

    public static void main(String[] args) {
        // Ejemplo de Pila
        Stack pila = new Stack<>();
        pila.push(1);
        pila.push(2);
        pila.push(3);
        System.out.println("Pila: " + pila);

        // Ejemplo de Cola
        Queue cola = new LinkedList<>();
        cola.add(1);
        cola.add(2);
        cola.add(3);
        System.out.println("Cola: " + cola);

        // Ejemplo de Lista
        LinkedList lista = new LinkedList<>();
        lista.add(1);
        lista.add(2);
        lista.add(3);
        System.out.println("Lista: " + lista);
    }
}
Resultado esperado: 
Pila: [1, 2, 3]
Cola: [1, 2, 3]
Lista: [1, 2, 3]
